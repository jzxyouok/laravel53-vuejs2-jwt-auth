## Install Step

    $ git clone https://github.com/imagine10255/laravel53-vuejs2-jwt-auth.git

    $ cd ./laravel53-vuejs2-jwt-auth.git

    $ composer install

    $ yarn install
    
    $ cp .env.example .env
    
@ 檢查 
    
1 根/.env 資料庫設定

2.如果你正在開發,請檢查 根/gulpfile.js browserSync.proxy: 'enter_your_hosting'
    
    $ php artisan migrate
    
    $ gulp watch
